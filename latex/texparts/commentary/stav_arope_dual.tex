Míra \ac{AROPE} v~\acloc{ČR} dosahuje v~roce~2024 hodnoty~\SI{11,3}{\percent}, čímž je nejnižší v~\aclloc{EU}27 (průměr~\SI{21,0}{\percent}). Ve~srovnávané šestici činí Polsko~\SI{16,0}{\percent}, Slovensko~\SI{18,3}{\percent}, Dánsko~\SI{18,8}{\percent}, Německo~\SI{21,1}{\percent} a~Rakousko~\SI{17,7}{\percent}; pozice \aclgen{ČR} je v~období 2015--2024 stabilně nejnižší.

Indikátor je vázán na~národní hranici chudoby (\SI{60}{\percent} národního mediánu disponibilního příjmu). Pro doplnění absolutního pohledu zobrazuje spodní panel obrázku horní hranici třetího decilu (\acs{var-D3}) ekvivalizovaného čistého příjmu v~\aclloc{PPS} jako podíl mediánu \aclgen{EU}27. \Acs{var-D3} \aclgen{ČR} dosahuje v~roce~2024 hodnoty~\SI{66}{\percent} mediánu \aclgen{EU}27; Rakousko \SI{111}{\percent}, Německo \SI{92}{\percent}, Dánsko \SI{92}{\percent}, Polsko \SI{67}{\percent}, Slovensko \SI{45}{\percent}~\cite{etui_cba}.

V~období 2015--2024 \acs{var-D3} \aclgen{ČR} vzrostl z~\SI{52}{\percent} na~\SI{66}{\percent} mediánu \aclgen{EU}27, Polsko z~\SI{49}{\percent} na~\SI{67}{\percent}, Slovensko klesl z~\SI{54}{\percent} na~\SI{45}{\percent}; Dánsko a~Německo poklesly o~\numrange{10}{13}~procentních bodů. Hodnoty mediánu \aclgen{EU}27 pro roky 2015--2017 jsou doplněny z~řady v~eurech (na~\aclloc{EU}27 platí $\mathrm{PPP}\equiv 1$); odchylka mezi řadami nepřesahuje~\SI{2}{\percent} na~překryvu 2018--2024.
